# Vault Learning Repository

This repository is my personal learning space for software development practice.

It contains hands-on projects, experiments, and notes focused on:

- C and C++ fundamentals and systems-level programming
- Java backend development with Spring ecosystem tools

The goal is to learn by building small, practical projects and iterating over time.

## Repository Purpose

I use this repository to:

- Practice language syntax and core programming concepts
- Explore frameworks and libraries in real code
- Compare different approaches to similar problems
- Keep a history of progress while learning

## Getting Started

### Prerequisites

Install the following tools:

- JDK 17+ (or the version required by each module)
- Maven (or use the included Maven Wrapper `mvnw`)
- A C/C++ toolchain (for example `gcc`/`g++` or `clang`/`clang++`)

## Notes

- This is a learning-first repository, so code style and project structure may evolve over time.
- Some modules may be experimental or incomplete while concepts are being explored.
