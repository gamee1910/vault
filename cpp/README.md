## Technologies

## Requirements

- C++ Compiler (GCC or Clang)
- CMake 3.10+

## Build and Run

### Using CMake:

```bash
cd cpp
mkdir -p cmake-build-debug
cd cmake-build-debug
cmake ..
make
```

### Or using IDE (e.g., CLion):
CMake configuration will be automatically detected.
